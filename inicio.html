BASE DE DATOS

Introducción

En esta página web vamos a ver el temario de la asignatura de base de datos hasta el dia de hoy. Veremos las 4 unidades dadas.

A continuación veremos en una tabla un resumen de las unidades. Usando el navegador de la derecha podrás indagar más en los temas que prefieras.

Unidad 1.

Almacenamiento de datos

 En esta unidad conoceremos los sistemas de almacenamiento y como empezar a crear una base de datos
Unidad 2.

Bases de datos relacionales

 En esta unidad trataremos con nuestro primer Gestor de Base de Datos, que será SQL, en concreto con MySQL.
Empezaremos con los tipos de gestión realizables. (DDL,DML,DCL,TCL). También veremos las sentencias CREATE, DROP y ALTER.
Unidad 3.

Realización de consultas

 En esta unidad, trataremos con sentencias de tipo DML para manejar Los datos que ya tenemos en nuestra Base.
Unidad 4.

Tratamiento de datos

 En esta unidad trataremos con sentencias de tipo DML pero esta vez para insertar o modificar datos de nuestra Base.
